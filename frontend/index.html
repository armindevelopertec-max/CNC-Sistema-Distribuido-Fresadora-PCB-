Del diseño Gerber a la fresadora CNC con GRBL

Esta plataforma simplifica el flujo completo: carga tus archivos Gerber, describe la configuración del PCB y observa el seguimiento del procesamiento. El backend organiza los archivos, valida formatos y prepara los datos del G-code, mientras la interfaz web guiada muestra cada etapa del sistema escolar.
