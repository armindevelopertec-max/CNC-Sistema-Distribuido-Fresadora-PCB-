Documentación del Sistema

Guía completa para el uso de la fresadora CNC de PCBs.

Arquitectura modular

Interfaz web -> servidor de aplicación -> CNC GRBL.

Interfaz web Carga Gerbers, configura el CAM y controla el flujo desde el navegador.
Servidor de aplicación Valida archivos, genera G-code y expone la API del sistema.
CNC GRBL Ejecuta las trayectorias y reporta el estado de la máquina.

1. Preparación de Archivos

El sistema requiere archivos Gerber estándar generados desde tu software de diseño (KiCad, Eagle, Proteus, etc.).

  • .gb1: Capa de pistas (Front Copper). Es el archivo obligatorio.
  • .gb0: Capa de contorno (Edge Cuts). Para el corte final de la placa.
  • .gb2: Capa de pads (Drill/Pads). Para perforaciones o marcas.

2. Flujo de Trabajo

  1. Importar: Arrastra tus archivos al panel de carga. El sistema los reconocerá automáticamente.
  2. CAM / Prep: Ajusta los parámetros de fabricación como profundidad, velocidad y diámetro de herramienta.
  3. Simulación: Verifica visualmente las trayectorias en el visor 3D antes de fabricar.
  4. Ejecución: Envía las capas una por una a la máquina CNC.

3. Parámetros Recomendados

Operación Profundidad Herramienta Avance
Trazas -0.06 mm 0.1 mm (V-Bit) 120 mm/min
Contorno -1.60 mm 0.8 mm (Fresa) 80 mm/min
Pads -0.06 mm 0.1 mm 120 mm/min
Nota de Seguridad: Asegúrate siempre de que el puerto serial esté conectado antes de iniciar la ejecución y mantén el botón de parada de emergencia a mano.
